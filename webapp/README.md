# WebApp متجر روز — المقتنيات

واجهة سوق مقتنيات فاخرة (RTL عربية، جوّال أولاً) تستهلك **واجهتنا فقط**
(`/api/v1/...`) ولا تتصل بـ MRKT إطلاقاً. تعمل داخل Telegram WebApp، وكموقع/تطبيق
مستقل مستقبلاً دون إعادة تصميم.

## الهوية البصرية: "فترينة المخمل"
قاعدة أوبرجين عميقة + لمسات ذهبي‑وردي (روز)، المقتنيات تُعرض كالجواهر في فترينة
مضاءة، وبطاقة السعر بلون الشمبانيا هي العنصر المميّز. تتكيّف مع سمتَي تيليجرام
الفاتحة/الداكنة مع بقاء هوية روز ثابتة.

## البنية (بلا خطوة بناء — ملفات ثابتة، ES modules)
```
webapp/
  index.html          الهيكل + تحميل Telegram SDK
  css/app.css         نظام التصميم الكامل (RTL، فاتح/داكن، متجاوب)
  js/
    config.js         عنوان الـ API الأساسي
    platform.js       طبقة المنصّة (Telegram | Web) — تعزل منطق تيليجرام
    api.js            عميل الـ API (يرفق initData للمصادقة)
    ui.js             العرض فقط: بطاقات/هياكل/تفاصيل/فلاتر/ترتيب
    app.js            المتحكّم: بحث/ترتيب/فلاتر/تمرير لانهائي
```

## المزايا
- شاشة رئيسية تعرض المقتنيات فوراً + شعار روز + بحث + فلترة + ترتيب.
- فئات ديناميكية (chips) من `/api/v1/categories`.
- بطاقات: صورة، اسم، مجموعة، رقم، شارة قفل، **السعر النهائي فقط** بعملة العميل.
- بحث فوري (اسم/مجموعة/رقم)، فلاتر (مجموعة/موديل/خلفية/رمز/مقفول/نطاق سعر)،
  ترتيب (الأرخص/الأغلى/الأحدث/الأقدم).
- تفاصيل المقتنى + زر **شراء** (placeholder فقط — لا شراء بعد).
- هياكل تحميل، صور كسولة، تمرير لانهائي، كاش صور، سمة فاتحة/داكنة.
- **لا يعرض إطلاقاً:** MRKT، اسم المزوّد، العمولة، GRAM، USD، معرّفات داخلية، بيانات خام.

## النشر
تُقدَّم الملفات عبر nginx (`deploy/nginx-collectibles.conf`) على HTTPS، ويُوجَّه
`/api` و`/media` للخدمة الداخلية. اضبط رابط الـ WebApp في BotFather على نطاقك.
عنوان الـ API افتراضياً نفس النطاق (`<meta name="roz-api-base" content="">`).
